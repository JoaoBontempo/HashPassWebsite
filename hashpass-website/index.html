Política de privacidade

Este aplicativo é mantido e operado por João Vitor Pedon Bontempo A privacidade e confidencialidade dos seus dados é de extrema importância para o HashPass, portanto, este documento tem como finalidade garantir a transparência do produto e transmitir informações sobre como os dados são tratados. Ao acessar, baixar, instalar e utilizar o aplicativo, você concorda com os seguintes termos abaixo 1. Dados utilizados no aplicativo Nenhum tipo de dado pessoal ou dado sensível será coletado explicitamente ou implicitamente em nenhum momento pelo aplicativo. 2. Armazenamento de dados Todos os dados inseridos pelo usuário no aplicativo são armazenados no próprio dispositivo, de forma local. Não há nenhum tráfego ou compartilhamento de dados pessoais ou sensíveis por rede para nenhum serviço externo ao aplicativo. Os usuários podem excluir as informações do aplicativo a qualquer momento. Os dados serão completamente excluídos caso o aplicativo seja desinstalado, tendo em vista que o armazenamento dos dados é realizado localmente. 3. Permissões O aplicativo somente necessita da permissão para utilizar o armazenamento (memória) do dispositivo para que seja possível utilizar a funcionalidade de exportação e importação de dados. Nenhuma outra permissão é necessária para utilizá-lo. 4. Medidas de segurança O aplicativo adota diversas medidas de segurança para garantir a confidencialidade das informações nele inseridas, das quais, destacam-se • Nenhum dado do usuário será trafegado em rede, todas as informações são armazenadas localmente; • Os dados salvos localmente são armazenados com criptografia simétrica; • A exportação de dados também utiliza métodos de criptografia; • A verificação de vazamento de senha utiliza hashes criptográficos, de forma que a senha real do usuário a ser verificada nunca é trafegada em rede. 5. Sobre as funcionalidades do aplicativo • Exportação e importação de dados É de responsabilidade do usuário garantir que o arquivo e a chave exportadas somente estarão disponíveis para pessoas autorizadas. O HashPass compromete-se em utilizar técnicas de criptografia para que, mesmo em casos de captura do arquivo ou da chave, a legibilidade dos mesmos esteja comprometida. Em nenhum momento, o HashPass se responsabiliza por qualquer vazamento de senhas ou informações do aplicativo por parte dos usuários. • Verificação de vazamento de senha O usuário terá total controle sobre tal funcionalidade, tendo a opção de utilizá-la ou não, de acordo com as configurações selecionadas por ele. A verificação é feita através de uma API pública que disponibiliza uma base da dados com senhas já vazadas na internet. Os dados trafegas com esta API são prefixos de hashes (algoritmo SHA-1) criptográficas das senhas. Portanto, em nenhum momento a senha será trafegada puramente em rede pelo aplicativo. 6.
